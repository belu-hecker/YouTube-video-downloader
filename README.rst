                                                      --------------------------------------------------
                                                            You Tube video & playlist Downloader
                                                      --------------------------------------------------

This is simple python file which will download any ``You TUbe video or playlist``. 

To run this file, you need to have pytube. To install it, run the following command in the terminal:

``pip3 install pytube``

To run the file, write the command:

``python3 downloader.py``

Follow-up your requirements to download your Video/playlist!
